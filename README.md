# 极客园

## 技术栈
* React
* sass
* antdDesign

## 项目结构
```
.
|-- src
|   |-- apis // 接口
|   |-- assets // 静态资源
|   |-- components // 通用组件
|   |-- pages // 路由组件
|   |-- router // 路由配置
|   |-- store // 全局状态管理
|   |-- utils // 工具函数

```
